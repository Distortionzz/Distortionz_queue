# 🚦 Distortionz Queue

**Distortionz Queue** is a premium timed connection queue for FiveM/Qbox servers.  
It gives every connecting player a short forced wait before entering the server, helping the server finish syncing resources/player data while showing a clean branded connection screen.

---

## ✨ Features

- 🚦 10-second timed connection queue
- 🎨 Premium Adaptive Card connection UI
- ⏳ Animated countdown
- 📊 Progress bar display
- 🔄 Rotating animated status messages
- 💡 Rotating server tips
- ✅ Final connection-approved screen
- 👥 Applies to all players equally
- 🧼 No priority queue or admin bypass
- ⚙️ Easy config customization
- 🧾 GitHub version-check support
